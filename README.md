# 🍔 Hungry Heist

**Hungry Heist** is a premium, full-stack food delivery platform built with the MERN stack (MongoDB, Express, React, Node.js). It features a modern, high-intensity UI tailored for the ultimate hunger quest.

## 🚀 Key Features

### 👤 Customer Experience
- **Smart Onboarding**: Secure authentication and detailed profile management.
- **Hunger Browsing**: Explore curated restaurants and masterfully crafted menus.
- **Risk-Free Cancellation**: A 60-second grace period for every order to ensure peace of mind.
- **Fair Fee Policy**: Automatic 100% cancellation fee recovery for late-stage cancellations, protecting our restaurant partners.
- **Live Heist Tracking**: Track your food from the kitchen to your door with live status updates.
- **Delivery Transparency**: Every heist records and displays the exact delivery destination.

### 🏪 Restaurant Partner Dashboard
- **Live Order Stream**: Manage incoming heists with real-time status updates (Accept, Prepare, Ready).
- **Proactive Rejection**: Ability to reject orders with specific reasons, keeping customers informed.
- **Revenue Analytics**: Track earnings and heist frequency through a dedicated partner dashboard.

### 🚴 Delivery Agent Command
- **Heist Assignments**: Efficiently manage and update the status of assigned deliveries.

## 🛠️ Technology Stack
- **Frontend**: React 19, Vite, Tailwind CSS 4.
- **Backend**: Node.js, Express 5.
- **Database**: MongoDB with Mongoose.
- **Icons**: FontAwesome 7.

## 📜 HEIST POLICY
1. **The 60s Window**: Cancellations within 60 seconds are free.
2. **The 100% Rule**: Cancellations after restaurant acceptance/60s result in a 100% fee, recovered automatically on the next checkout.
3. **Daily Limit**: For platform integrity, customers are limited to 3 free cancellations per 24-hour window.
